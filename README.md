# QuCIS Lab – Quantum Computing & Information Systems

A high-fidelity, modern landing page for the **Quantum Computing and Information Systems (QuCIS) Lab** at IISER Bhopal, led by **Dr. Ankur Raina**.

This project is a complete redesign of the lab's web presence, focusing on a "Kimi Agent" inspired dark theme with interactive quantum-themed animations and a seamless user experience.

## ✨ Features

- **Interactive Quantum Background**: A high-performance HTML5 Canvas animation featuring "Orbital Spheres" that react to mouse proximity, symbolizing quantum states and entanglement.
- **Dynamic Content Sections**:
  - **About**: Detailed overview of lab research and stats.
  - **Team**: Categorized directory of PhD, BS-MS, and Project students, plus an interactive Alumni filter by year.
  - **Publications**: Organized list of Journals, Conferences, Preprints, and Posters.
  - **Student Showcase**: A visual gallery of student projects with modal carousels and detailed descriptions.
  - **Funding**: Logo grid of primary funding agencies.
  - **Careers**: Opportunities for prospective students and researchers.
- **Modern UI/UX**:
  - Glassmorphic navigation bar with seamless blending.
  - Responsive design for mobile, tablet, and desktop.
  - Dark-themed aesthetic with vibrant cyan and orange accents.
  - Smooth scroll and instant navigation between dedicated pages.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Version 22.x or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/sections/`: Core UI sections (Navbar, Hero, About, Team, etc.)
- `src/pages/`: Dedicated page components for routing.
- `src/components/`: Reusable UI components and wrappers.
- `src/data/`: Structured data files for members, publications, and showcase.
- `public/`: Static assets (Logo, Hero video, project images).
